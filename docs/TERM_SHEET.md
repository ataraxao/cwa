## Operative Date / Condition Precedent
The Agreement is **not operative** until the structure is actually implemented and the Independent Administrator issues written **Implementation Confirmation**.

### Required controls
- No unilateral transfers-out by either party
- Independent Administrator controls transfers-out / distributions / freeze actions for Administrator Events
- Party A has trading authority only
- Permission-change controls restrict unilateral payout-destination / linked-account changes
- Party B has view / statement access only, if any
- The Administrator has reasonably sufficient operational records confirming the control split is actually recognized by the relevant provider

### Non-operability consequence
- If the Agreement never becomes operative, no Allocation Percentage, Liquidity payout, joint release entitlement, or Termination payout arises by implication, partial implementation, account activity, or reliance
